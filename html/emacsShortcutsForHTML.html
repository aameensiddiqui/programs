There are many shortcuts in emacs for creating html tags.

Like:

  • C-c RET......... p
  • C-c 1........... h1
  • C-c 2........... h2
  • C-c 3........... h3
  • C-c 4........... h4
  • C-c 5........... h5
  • C-c 6........... h6
  • C-c C-j......... vertical line break
  • C-c C-c -....... horizontal line break
  • C-c /........... closing a tag
  • C-c C-c u....... unordered list
  • C-c C-c o....... ordered list
  • C-c C-c l....... list item
  • C-c C-c #....... id
  • C-c C-c n....... name
  • C-c C-c h....... href
  • C-c C-c i....... image
  • C-c C-c......... checkbox
  • C-c C-c r....... radio button
  • C-c C-o......... insert tag
  • C-c C-a......... insert attributes
  • C-c C-f......... forward tag
  • C-c C-b......... backward tag
  • C-c Backspace... delete tag
  • C-c C-o ........ search the tag you don't know about
  • C-c TAB ........ toggle visibility of a tag
  • M-o u........... underline
  • M-o b........... bold
  • M-o i........... italic
  • C-c C-d ........ segment delete tag
  • C-c C-e 2....... segment close tag
  • C-c /........... segment close tag
  • C-c ]........... segment close tag
  • C-c ...... segment skip tag backward
  • C-c ..... segment skip tag forward
  • C-c C-;......... comment
